CSS TOPIC - BORDERS WITH ROUNDED CORNERS

Using CSS anyone can make borders appear around their HTML elements.

The first step is add a border to your element. You can do this with: "border-style: solid;"

Note: border-style has many options for the type of line you want. You can make it solid, dashed, dotted, groove, etc.

Note: you can also add color to your borders. Add border-color: Black; to make it black. Swap out the black with any color option you would like.
